# Translator-App
Django Translator App
A simple and user-friendly language translation web application built using Django and Googletrans. 
This project by jk2004-ai allows users to enter text, select a language, and get instant, accurate translations. 
Perfect for beginners learning Django, Python, or working with APIs.

Features
1.Translate text instantly into multiple languages
2.Clean and responsive interface
3.Uses Googletrans (free translation API)
4.Django-powered backend
5.Error handling for missing translation cases
6.Easy to customize & expand

 Tech Stack

1.Python
2.Django Framework
3.Googletrans API
4.HTML & CSS
5.Virtual Environment (venv)

How to Run the Project

1.Clone the repo
clone https://github.com/jk2004-ai/translator-app.git
cd translator-app
2.Activate virtual environment
python -m venv venv
venv\Scripts\activate   (Windows)
source venv/bin/activate (Linux/Mac)
3.Install requirements
pip install -r requirements.txt
4.Run the server
python manage.py runserver
5.Open in browser:
http://127.0.0.1:8000/
